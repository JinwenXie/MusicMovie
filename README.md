### 简介
这是一个通过豆瓣电影API来获取到电影数据的微信小程序,**非常适合作为小程序初学者实战项目！**

### 进程

1. 正在热映电影首页（已完成）；
2. 即将上映电影首页（已完成）；
3. 电影排行榜首页（已完成）；
4. 电影搜索页（暂未完成）；
5. 正在热映电影列表页（暂未完成）；
6. 即将上映电影列表页（暂未完成）；
7. 电影排行榜列表页（暂未完成）；
8. 电影详情页（暂未完成）；
